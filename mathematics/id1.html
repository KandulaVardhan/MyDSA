Problem - Count number of digits in an integer
Approach-1: Using a while loop to count each digit and using division operator to remove the last digit each time
Time Complexity- O(logn), Space Complexity- O(1)
Steps:
  • 'n' as user input, 'ans' = 0
  • while checking the loop, if n is not 0, satisfied=> remove the last digit and increment ans
  • once the loop is broken, return the ans
Approach-2: Using log to the base 2
Time Complexity- O(1), Space Complexity- O(1)
